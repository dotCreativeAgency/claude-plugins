# code-explainer

Claude Code plugin that explains code with visual ASCII diagrams, everyday analogies, step-by-step walkthroughs, and common gotchas.

## Installation

```bash
# Add the marketplace
/plugin marketplace add dotCreativeAgency/claude-plugins

# Install the plugin
/plugin install code-explainer@dotcreativeagency-plugins
```

## Skills

### explain-code

Activates when explaining how code works, teaching about a codebase, or when the user asks "how does this work?"

Every explanation includes:

1. **Analogy** - Compares the code to something from everyday life
2. **ASCII diagram** - Visual representation of flow, structure, or relationships
3. **Step-by-step walkthrough** - What happens and in what order
4. **Gotcha** - Common mistakes or misconceptions
